# 2026-04-04

- **v5.4.10** — Fix 189,518 orphaned photo_tags from v5 migration (missing CLO/LO pointers), regenerate 170,992 stale summaries, reprocess 1,041 XP corrections. Three new artisan commands: `olm:fix-orphaned-tags`, `olm:regenerate-summaries`, `olm:reprocess-metrics`. Production runbook at `readme/changelog/production-orphan-fix-runbook.md`.

- **v5.5.1** — Global map tab title now shows live event count (e.g. `OpenLitterMap - (3)`), incrementing on new events and decrementing on dismiss.
- **v5.6.0** — Fix impact report tweet publishing: switch brands query from deprecated v4 schema to v5 `photo_tag_extra_tags`, make Chromium path configurable via `BROWSERSHOT_CHROME_PATH` env var, add new annual impact report command (`twitter:annual-impact-report-tweet`) scheduled Jan 1 at 06:30, add annual period support to `GenerateImpactReportController`, 8 new tests.
- **v5.6.1** — Impact report cleanup: use `Timescale` enum instead of magic integers, fix cache TTL bug (past periods were never cached), make annual `getMetricsRow` branch explicit, simplify `mkdir` in all three Browsershot commands, add `waitUntilNetworkIdle()` to weekly command.
