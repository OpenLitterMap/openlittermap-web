## Session: Public most-tagged litter statistic

- v5.12.5 ‚Äî Add `--scope` option to `olm:rebuild-tag-counts`: `total` (default/omitted) is byte-for-byte the existing behaviour writing `resources/data/tag_usage_counts.json` (scope `total_recorded_tags`); `public` adds `WHERE is_public = 1 AND verified >= 2` and writes a separate committed file `resources/data/tag_counts_public.json` (scope `verified_public_on_map`) at the same (object, category, type) granularity so the two are diffable row-for-row.
